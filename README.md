## yarn
https://yarnpkg.com

<br />

## Criação do projeto

Criar o diretório para o projeto
```shell
$ mkdir nome_do_projeto
$ cd nome_do_projeto
```

Iniciar com o yarn
```shell
$ yarn init -y
```
<br />

## Adição de bibliotecas
## express
```shell
$ yarn add express
```
<br />

## Execução
Rode o comando do terminal no diretório raiz do projeto
```shell
$ node index.js
```
Acesse o endereço no navegador
http://localhost:3000/hello
